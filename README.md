# 📊 Power BI Dashboards Portfolio

This repository contains a collection of interactive dashboards built using Microsoft Power BI. Each project focuses on real-world business scenarios including manufacturing analytics, retail performance, and delivery time estimation.

---

## 🚀 Tools Used

* Microsoft Power BI
* Data Modeling & Transformation
* DAX (Data Analysis Expressions)
* Data Visualization Best Practices

---

## 📁 Projects Included

### 🏭 1. Manufacturing Overview Dashboard

**Description:**
This dashboard provides insights into machine performance and sales in a manufacturing environment.

**Key Features:**

* Machine utilization and efficiency tracking
* Production vs downtime analysis
* Sales performance by product/category
* KPI indicators for operational monitoring

---

### 🛍️ 2. Retail Sales Dashboard

**Description:**
A comprehensive analysis of retail sales data to uncover trends and improve decision-making.

**Key Features:**

* Sales trends over time
* Top-performing products and regions
* Customer purchase behavior
* Revenue and profit breakdown

---

### 🚚 3. Delivery Time Estimation Dashboard (Blinkit-style)

**File:"Sixth.pbix"

**Description:**
This dashboard simulates estimated delivery time analysis similar to quick-commerce platforms operating in China.

**Key Features:**

* Estimated vs actual delivery time
* Order volume and peak time analysis
* Location-based delivery performance
* Factors affecting delivery delay
